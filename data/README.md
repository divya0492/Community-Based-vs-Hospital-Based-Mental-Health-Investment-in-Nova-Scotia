# Overview
<p align="justify">
The data for this project was sourced from the Canadian Institute for Health Information (CIHI) Your Health System Indicator Library, downloaded as two Microsoft Excel files (dataset1.xlsx and dataset2.xlsx), both containing identical Nova Scotia health indicator data spanning approximately 48,002 rows and 100 indicators across multiple health domains.
From this full dataset, the data was filtered to retain only indicators directly related to mental health. Four indicators were identified and extracted: Canadians With a Mental Health Disorder Who Have an Unmet Need for Mental Health Care, Self-Harm Hospitalizations, Self-Harm Including Suicide, and Wait Times for Community Mental Health Counselling. 
<p align="justify">
The filtered dataset, comprising 616 rows, was then exported into a Tableau Hyper extract file dataset3.hyper for visualization purposes. This file served as the primary data source for building the Tableau dashboard and all associated visual outputs for the project.
</p>
